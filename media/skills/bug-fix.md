# Bug Fix Skill

Use this workflow when fixing bugs.

Focus on:
- Reproduction
- Root cause
- Minimal fix
- Regression tests
- Edge cases

Rules:
- Identify the likely root cause before editing.
- Keep the patch scoped.
- Avoid unrelated cleanup.
- Add or update tests when possible.

Output:
- Bug summary
- Root cause
- Fix plan
- Files to change
- Tests to add
